route(url, handler, relativeToBaseUrl)

Route to url, or register a handler to respond to url. If handler is passed: registers handler as a route for the given url pattern. If handler is omitted: Attempts to route to the given url.

Parameters:
Name Type Argument Default Description
url String

A URL to route to.
handler function <optional>

If given, url is instead used as a pattern that maps URLs to handler.
relativeToBaseUrl Boolean <optional>
 true

If false, the router will attempt to route to url without taking baseUrl into account.
Source:
  • router.js, line 109
Examples

Registering a route handler:

router.route('/jobs/:id', function(params) {
	console.log('Job ID: ' + params.id);
});

Invoking a route: (uses previous example's route handler)

var didRoute = router.route('/jobs/1337');
console.log( didRoute );   // true
